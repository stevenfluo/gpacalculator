High School Grade Point Average Calculator

About the Grade Point Average (GPA)

Students receive 1 set of grades per semester, 2 per year. Grades of P for Pass or NP for Not Passed are not calculated into the GPA. GPA can be either weighted or unweighted. The main difference between two types of GPA is that the weighted GPA includes an extra point for AP courses.

All GPAs are calculated by assigning each letter grade a number value according to the graph listed below. The total sum of grade points is then divided by the number of classes in the calculation. Some teachers use pluses and minuses (+ or -) in the reporting of grades, however it is not calculated into the GPA.

Using the GPA Calculator

You will need your report card or transcript to use this calculator.

Count the number of A grades you've received in regular (non-AP/IB/Honors) classes. Input that number into the first row. Repeat this process for each of the letter grades.

In the "AP/IB/Honors Classes" Section, count the number of A grades you've received in AP/IB/Honors classes. Input that number into the first row. Repeat this process for each of the letter grades.

Once you are finished, click on the "Calculate" button or press the "Enter" key on your keyboard to display your Weighted and Unweighted GPAs. To reset all values back to 0, click on the "Reset" button.
